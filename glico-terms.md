## Description of Service

Glico is a blood glucose tracking application that helps users log and monitor their glucose readings through:
- Manual blood glucose logging with context and notes
- Live dashboard with daily averages, last value, range, and on-target percentage
- Detailed logbook with history and analytics
- Custom target ranges per measurement context
- Home screen and lock screen widgets
- Smart reminders and notifications
- Optional iCloud synchronization via CloudKit
- Apple Health integration via HealthKit
- CSV export for sharing data with healthcare providers
